# Omni Analytics Portfolio

A portfolio of data modeling, dashboard design, and dbt integration work built on Omni Analytics and MotherDuck.

**Full documentation coming soon.**

## Projects
- Phase 1: Bigfoot Sightings & UFO Sightings
- Phase 2: NYC 311 Service Requests & Stack Overflow Developer Survey
- Phase 3: NHTSA Vehicle Complaints & CDC Cardiovascular Disease Indicators
- Phase 4: dbt Core integration with NYC 311 data (staging + mart models)

## Tech Stack
- **BI Tool:** Omni Analytics
- **Data Warehouse:** MotherDuck (DuckDB)
- **Transformation:** dbt Core + dbt-duckdb adapter
- **Languages:** SQL, Python
